Online Form

This is an online form with all the typical input fields.

Successes

This was a relativley straight-forward project that I enjoyed. Easy layout and architecture.

Challenges

  • Zipcode dropdown menu was a new element for me at this point.
  • Differences between the variety of attributes like "id", "name", "for", "type" etc.

Adapt and Overcome

Taking thorough notes, referencing resources (books and online sites), leaving comments in my code and using an html validator was a fool proof way to grasp the concept of form attributes.

I also learned the importance of order, label tags before input tags. Little things like this can be overlooked but every detail is important.
