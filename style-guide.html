SASS Web Style Guide

This is an basic example of a useful template for styling web pages.

Successes

This was the first project to use SASS and taught me much more about CSS than I thought I knew. This project took longer than the estimated 10 hours, but I have been applying SASS to every project since this one to further my understanding.

Challenges

  • Initial setup of SASS and saving files successfully.
  • Preset classes caused stylings on other elements that I did not intend to style.
  • Differences between and when to use mixins and extends

Adapt and Overcome

It was easy to mispell or mislabel files and this caused errors. To solve these errors, its important to start from the beginning. It helps to step away and come back with fresh eyes.

Placing colored borders around the pre-set classes I was able to create a visual representation of all the elements that shared classes. This helped with styling later.

This project taught me that the learning never stops, there will always be something I don't know or understand. I got comfortable searching other resources for tutorials and information to add to my collection of notes and lessons learned.
