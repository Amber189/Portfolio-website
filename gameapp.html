Wheel of Success

This is a web style game app like Wheel of Fortune and uses JavaScript.

Successes

Used console.log to check results of code and display answer for testing. Added transitions on letters for extra credit.

Challenges

  • Split the phrase into characters and display them in a list item, taking spaces into consideration
  • Button reveal would only reveal one letter even if there were more.
  • If there are 5 "misses" reveal a "game lost" overlay, otherwise, display "winner" overlay.

Adapt and Overcome

Splitting the phrase took a bit of playing around. Discovered a method that counts for spaces.

A return statement inside of my "if" statement was throwing errors. Removed it and made sure all letters were made lowercase before comparing values.

Went to a meetup to seek face-to-face advice. I didn't realize there was an overlay already coded into the css provided and discovered my if statement was not counting the number of "misses" correctly.
